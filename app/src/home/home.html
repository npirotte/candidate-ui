Description

The project is split in two part :

Frontend

Tools/Frameworks : NodeJS, Bower, AngularJS, Bootstrap, HTML, CSS, etc...
Remarks : All requests are mocked, no backend needed.

Backend

Tools/Frameworks : Java 8, Spring Boot, Hibernate, Gradle, etc...
